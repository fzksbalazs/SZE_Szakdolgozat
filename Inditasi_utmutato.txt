A projekt összes egysége egyetlen kattintással elindítható:

Kattintson duplán a „start_all.bat” fájlra.

Ez automatikusan megnyit 6 külön parancssori ablakot, és elindítja a
következő szolgáltatásokat:

-   Admin felület: http://localhost:3001
-   Webshop frontend: http://localhost:3000
-   Backend API: http://localhost:5000
-   T-shirt designer kliens: http://localhost:5173
-   T-shirt designer Node.js szerver: http://localhost:8080

Python szerver futtatasa: (első alkalommal szükséges)
•	Telepítsd a Python 3.11.9 verzióját, A telepítőben jelöld be Add Python to PATH.
•	Script futtatás engedélyezése PowerShellben: Ha ezt írja: "running scripts is disabled”: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
•	A projekt virtuális környezetének (.venv) létrehozása:- Navigálj a backend AI mappához: cd SZE_Szakdolgozat-main\tshirt_designer
o								                          -Hozd létre a .venv-et: python -m venv .venv
•	A. venv aktiválása: --> .venv\Scripts\activate  -->a tshirtdesigner mappában
•	Kötelező csomagok telepítése (requirements.txt): python -m pip install -r requirements.txt
•	uvicorn szerver indítása: cd SZE_Szakdolgozat-main\tshirt_designer\server\python --> uvicorn app:app --host 0.0.0.0 --port 8000
